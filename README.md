# Eduford University Website

A modern, responsive university website built with pure HTML, CSS, and JavaScript. This project demonstrates front-end development skills suitable for a junior full-stack developer portfolio.

## Live Demo

Open `index.html` in your browser to view the website.

## Features

- **Responsive Design**: Fully responsive layout that works on desktop, tablet, and mobile devices
- **Multi-page Structure**: 5 interconnected pages (Home, About, Courses, Blog, Contact)
- **Modern UI/UX**: Clean, professional design with smooth animations and transitions
- **Mobile Navigation**: Hamburger menu with smooth slide-in animation for mobile devices
- **Interactive Elements**: Hover effects, form validation, and smooth scrolling
- **Accessibility**: Semantic HTML, alt tags, ARIA labels, and keyboard navigation support
- **SEO Optimized**: Meta tags, descriptions, and structured content

## Pages

1. **Home** (`index.html`) - Landing page with hero section, courses overview, campus showcase, facilities, and testimonials
2. **About** (`about.html`) - University history, mission, and values
3. **Courses** (`course.html`) - Detailed course offerings and facilities
4. **Blog** (`blog.html`) - News, updates, and educational content
5. **Contact** (`contact.html`) - Contact form with validation and location map

## Tech Stack

- **HTML5** - Semantic markup and structure
- **CSS3** - Flexbox, Grid, animations, and responsive design
- **JavaScript** - DOM manipulation and mobile menu functionality
- **Font Awesome** - Icons and social media links
